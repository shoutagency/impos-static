POS Software

Back Office

Access your system from anywhere in the world and make changes – instantly!

Saving a product in ImPOS is very fast and straight-forward. Positioning the product on the screen involves selecting the product and dragging it into the desired position with all screen designs being updated simultaneously. Multiple menu designs can be managed from the one screen ensuring speed on the front end and allows customisation to suit different areas of the business.

Previous slide Next slide

Define Buttons

The ImPOS menu design is completely graphical - there is no reference to PLU numbers, columns or rows. What you design is what you get and it is all designed by selecting your requirements and dragging them into position. All changes are instantly updated to the terminals with no need for restarting the programs. Define buttons are also used to configure the layout of ImPDA. From the define buttons screen you can define anything that is required to be on the screen including:

  • Items
  • Modifiers
  • Adjustments
  • Sub List
  • Pay Method
  • QuickPay
  • Function
  • Discount
  • Set-Menu

Multiple EPOS menu lists can be accessed from the define buttons screen ensuring flexibility and speed on every transaction.

Item Setup

  • Instant search function
  • Edit existing items
  • Assign to departments
  • Up to fifteen different price levels per item
  • Happy hour or time based pricing
  • Five print settings
  • Stock count-down
  • GPM cost input
  • Picture/Logo
  • Ingredient make-up
  • Barcode integration - great for wineries
  • Set-Menu

Sub lists

  • Price Set overview - you may define a sub-list to look at different price sets for items great for functions or specials items
  • Course overview - as soon as an entrée item is selected it can be grouped under the entrée course grouping in the kitchen
  • Modi-List setup - quickly define the modifiers that can be attached to items, e.g. steak - cooking method - sauces
  • Define discounts

Auto Modifier

Most EPOS will allow you to allocate modifiers for an item, ImPOS allows you to set rules for the modifiers selected within a list which is effective for venues that serve mains with a selection of two side dishes for example - this way the staff can't select a main without selecting two side dishes.

Cooking instructions

Using the Auto Modifier feature you can allocate what items are to be included for a particular dish. This function was developed to enable preparation instructions to be added automatically to any menu item.

Table Layout

Managing different layouts with ImPOS’ superior POS software is easily achieved. With the ability to configure multiple layout designs, e.g. weekend/weekday or function A, B, C and load a layout at any given time. Different areas within the venue can be setup with floor plan layouts/images sitting underneath the tables to familiarise staff with the table positions.

Time based Rules

Defining price changes based on time of the day or entire days such as public holidays is a simple process that takes seconds to configure. The flexibility of 15 different price levels per item allows multiple different price changes to be made efficiently and effortlessly.

Setting up users/employees

Each user or employee can be defined to a particular department thus enabling a cost vs. department report to be created with an hourly rate defined per employee. Smart Codes (wrist bands, swipe cards etc.) can be allocated to each employee for added security. Up to 100 different authorisation levels can be defined when entering or editing an employee's attributes. This enables the administrator to lock an employee out of a particular function such as voiding or refunding.

Employee Messages

Messages can be sent to employees instantly or setup so that the message appears on screen at a particular date/time in the future. When the message is read it records this with the date and time that the employee has read the message. This POS software feature was designed to assist with communication within the business e.g. rostering or reminding employees of upcoming events and promotions.

Accounts/Payment methods

Defining an account is fast and easy with full account transaction reporting details available. You have full control over your accounts and the web-based reports have the capability to filter, print and save any account report from off-site.
